Dashboard Deep-Dives
1. Sales Analysis Dashboard
Designed for the Sales department to monitor revenue health and profitability trends.

Seasonality: March and April 2014 showed peak revenue, while December recorded the lowest sales.

Filtering Capabilities: Allows users to drill down by Year, Quarter, Category, and Order Priority.

Recommended Filter: Year: 2014 | Category: Technology | Order Priority: Medium.

2. Product Analysis Dashboard
Focuses on inventory movement, quantity demand, and pricing strategies.

Volume Leader: Office Supplies recorded the highest quantity ordered across all categories.

Discount Strategy: In 2014, Tables were sold at an average discount of 28%.

Market Insight: Deep-dive filters available for Year (2014) and Market (US).

3. Shipping Analysis Dashboard
Analyzes logistics efficiency and the relationship between order priority and shipping modes.

Preferred Modes: Standard Class is the most utilized mode (53% of orders), while Same Day is the least preferred.

Priority Correlation: 100% of "Low Priority" orders and 41% of "Critical" orders were fulfilled via Standard Class and First Class respectively.

Cost Analysis: APAC markets incur the highest shipping costs, followed by the EU and US.

Tools Used
Microsoft Power BI: Data Modeling, DAX (Calculated Measures), and Visual Storytelling.

Power Query: For data cleaning and transformation (ETL).

How to Use This Repository
View Screenshots: Open the /Dashboards folder to view high-resolution captures of the Sales, Product, and Shipping reports.

Interact: Download the .pbix file to explore the interactive slicers and drill-down features.

Read Insights: Refer to this README for a summary of high-level organizational trends.
